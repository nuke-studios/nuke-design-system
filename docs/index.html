NUKE

Two variables control everything. One unit. One ratio. Pure math.

Configuration

--unit 6px Base size
--ratio 0.4 Content fill

All other tokens derive from these two values.

Presets

  • 4px Compact
  • 6px Standard
  • 8px Spacious

Features

Semantic HTML

Native elements with nuke-style attributes. No framework lock-in.

Token System

Modular scale. Change --unit and everything adjusts.

Themes

Dark and light. Add .light-theme to :root.
